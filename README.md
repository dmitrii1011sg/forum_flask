## Начало работы

Что бы запустить сайты выполните следующие простые шаги.

### Установка

1. Клонируйте репозиторий
   ```sh
   git clone https://github.com/dmitrii1011sg/forum_flask.git
   ```
2. Установите все библиотеки
   ```sh
   pip install -r requirements.txt
   ```
